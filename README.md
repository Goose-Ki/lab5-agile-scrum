# Лабораторная работа №5 — Agile и Scrum

Football Match Tracker — учебное приложение для отслеживания футбольных матчей.
Проект позволяет просматривать команды, матчи и их результаты.
Цель работы — изучить Agile-подход и Scrum-процесс на практике.
Разработка организована с использованием GitHub Issues.
Все требования оформлены в виде product backlog.
В рамках лабораторной работы смоделирован один Scrum-спринт.
